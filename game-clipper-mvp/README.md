# GameClipper MVP

ゲーム配信者向けの自動切り抜きツールMVPです。

## 機能

- 動画ファイル読み込み
- 音量ピークから盛り上がりシーンを検出
- 候補のプレビュー
- FFmpeg.wasmでMP4切り抜き書き出し
- タイトル風テキスト生成

## 使い方

1. `index.html` をブラウザで開く
2. 動画を選択
3. 「盛り上がりシーンを検出」を押す
4. 候補をプレビュー
5. MP4を書き出し

## 注意

ブラウザで直接開いた場合、環境によってFFmpeg.wasmが動かないことがあります。  
その場合はローカルサーバーで起動してください。

```bash
python -m http.server 8000
```

その後、以下を開きます。

```text
http://localhost:8000
```
